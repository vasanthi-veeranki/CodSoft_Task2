# Movie Rating Prediction

## About the Project

This project was completed as part of the **CodSoft Data Science Virtual Internship**. The objective is to build a machine learning model that predicts movie ratings based on various features from the dataset. The project demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis, feature engineering, model training, and performance evaluation.

## Technologies Used

* Python
* Visual Studio Code (VS Code)
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Project Workflow

* Data Collection
* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Model Training
* Model Evaluation
* Rating Prediction

## Outcome

This project enhanced my understanding of regression techniques, data analysis, and predictive modeling using machine learning.
